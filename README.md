# Lounge21 Website Skeleton

This is a minimal Vue 3 + Vite skeleton for the Lounge21 site.

## Tech stack

- Vue 3
- Vue Router 4
- Vite
- SCSS
- ESLint (Vue 3 recommended rules)

## Structure

- `src/main.js` – app entry, global styles
- `src/App.vue` – root component using `TheMainLayout`
- `src/router/index.js` – routes for core pages and a creator placeholder
- `src/views` – page components (Home, About, Contact, Legal, Creators)
- `src/components/layout` – shared layout: header, footer, shell
- `src/components/ui` – basic UI elements (button, card, section)
- `src/styles` – tokens, base styles, and main stylesheet

Run:

```bash
npm install
npm run dev
```

and start building out the Lounge21 experience.
